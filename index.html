What we do


  • Effective Machine Learning solutions comprise of many different parts which must compliment each other to achieve overall accuracy and performance. This includes the right hardware, the models, the pipeline and of course the data sets. Poor data set selection, wrong pre-processing and labeling can render any ML-system useless and let all the effort of setting it up go to waste. At the same time, data set annotation is a tedious process which require certain level of understanding how ML works for a specific project but at the same time it is unreasonable to be performed by high skilled ML engineers.





  • We fill this niche. Coming from a vast experience working with multiple ML projects we have build an effective and precise process for data processing. Aided by experienced ML engineers, our annotators do the best job possible to create data sets which would make the most of any model you have come with.


  • It is not enough to build a good supervised ML system: you have to teach it. And this is precisely what Ratatoskr is about. Once you have settled on a model, we take care of the data sets to achieve the best learning performance. Coverage recommendations, selection and balancing, pre-processing and annotation, post-processing and supplying in the right format - as long as data sets are concerned, we have it covered.
